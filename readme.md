User can delete the task by clicking on the delete icon.

- Thinking about the UI
- Functionality

	- Delete a TODO

- Data
	- tasks - an array
	- task - {done, text, id}
- Functions (in code)
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
