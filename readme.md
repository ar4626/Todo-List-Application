### General steps to follow when creating a project

- Thinking about the UI
- Functionality

	- Delete a TODO

- Data
	- tasks - an array
	- task - {done, text, id}
- Functions (in code)
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
